# Corelab API

## Using

- Express.js 4.18.1

- Typescript 4.7.4

- Dotenv

- ODM: Mongoose 6.4.3 to MongoDB (Atlas - GCP/Amazon/Azure)

- CORS

- Prettier

- Eslint

## How to execute

### Clone

`git clone https://github.com/leandrocunha526/corelab-api`

### Install dependencies

`npm install`

### Execute

To execute use `npm run dev` and set your MONGODB_URI em .env.
